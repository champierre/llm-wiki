# TMPose2Scratch

Google Teachable Machine 2のポーズ認識とScratchを接続するScratch 3拡張機能（GitHub ★6）。[[entities/ml2scratch|ML2Scratch]]から派生したプロジェクトの一つ。

## 概要

Google Teachable Machine 2で作成したポーズ認識モデルをScratch 3で利用できるようにする拡張機能。Teachable Machineのポーズプロジェクトで訓練したモデルの認識結果をScratchプログラムに取り込むことで、体の動きに基づく機械学習を活用した作品制作を可能にする。

## 技術

- **言語**: JavaScript
- **ML**: Google Teachable Machine 2（ポーズ認識）
- **プラットフォーム**: Scratch 3
