# TM2Scratch

Google Teachable MachineとScratchを接続するScratch 3拡張機能（GitHub ★31）。[[entities/ml2scratch|ML2Scratch]]から派生したプロジェクトの一つ。

## 概要

Google Teachable Machineで作成した機械学習モデルをScratch 3で利用できるようにする拡張機能。Teachable Machineの直感的なGUIでモデルを訓練し、その認識結果をScratchプログラムに取り込むことで、機械学習を活用した作品制作をより手軽にする。

## 技術

- **言語**: JavaScript
- **ML**: Google Teachable Machine
- **プラットフォーム**: Scratch 3
