# 横川耕二（Koji Yokokawa / Yokobond）

横浜在住のフリーランスのコンピュータープログラマ、プロトタイプエンジニア、教育者。青山学院大学 [[entities/tukumana-lab|青学つくまなラボ]]プロジェクト教員（准教授）。「人のために役に立つ、ユニークで楽しいコンピューターシステムを作ること」を目指す。

Scratchとハードウェアを接続する**フィジカルコンピューティング**の領域で、石原淳也の**機械学習×Scratch**と並ぶもう一つの柱を形成している。

## 主要プロジェクト

### Microbit More
Scratchの標準micro:bit拡張機能の制限を超え、micro:bitのすべてのセンサー・LED・デジタル/アナログ入出力をScratchから利用可能にする拡張機能。横川耕二の代表的なプロジェクト。

- **Web**: https://microbit-more.github.io/index-ja.html
- **対応**: micro:bit v1 / v2
- **特徴**: オープンソース、標準拡張では使えないセンサー群をフルに活用可能
- **活用例**: PCと紙工作を組み合わせた射的ゲーム、風鈴、気象・運動などの物理現象の可視化
- [[entities/tukumana-lab|青学つくまなラボ]]のワークショップ「Takorach対応ピンポンゲーム」でも使用されている

### Squeak Etoys + GAINER / Scratch + Arduino ライブラリ
子ども向けプログラミング環境（Squeak Etoys、Scratch）にお手軽I/Oデバイス（GAINER、Arduino）を接続するライブラリ。誰でもすぐに簡単にフィジカルコンピューティングができる環境を実現。2009年のMake: Tokyo Meeting 03/04で発表。

## 石原淳也との関わり

### つくまなラボでの協働
[[entities/tukumana-lab|青学つくまなラボ]]で共にフェロー（総合文化政策学部プロジェクト准教授）を務める同僚。

### Scratch拡張の「二つの方向」
石原淳也とは、Scratchを拡張するという同じ志向を共有しつつ、異なる方向に特化している：

| | 石原淳也 | 横川耕二 |
|---|---|---|
| **方向** | AI・機械学習 × Scratch | ハードウェア・フィジカルコンピューティング × Scratch |
| **代表作** | [[entities/ml2scratch|ML2Scratch]]（★120） | Microbit More |
| **技術** | TensorFlow.js, MediaPipe | micro:bit, GAINER, Arduino |
| **命名** | 〇〇2Scratch | Microbit More |

両者は[[entities/scratch-extensions|Scratchエコシステム]]の中で補完関係にある。実際にFacebookの投稿では「石原淳也さんのML2Scratch（画像認識機械学習）と横川耕二のMicrobit More」が並べて言及されている。

### 共通の人脈
- **[[entities/abe-kazuhiro|阿部和広]]**: 横川はNHK Eテレ『Why!? プログラミング』で阿部和広（プログラミング監修）のもとScratchプログラム制作を担当。IPA未踏プロジェクトでもアラン・ケイ博士のもとで開発を行っており、阿部和広と同じくアラン・ケイの系譜に位置する。
- **Maker Faireコミュニティ**: 両者ともMaker Faire Tokyoに出展経験がある。
