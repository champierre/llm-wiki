# 阿部和広（Abe Kazuhiro）

1965年生まれ、倉敷市出身。青山学院大学特任教授、放送大学客員教授。Scratch教育の日本における第一人者。[[entities/tukumana-lab|青学つくまなラボ]]のシニア・フェロー。アラン・ケイ博士に師事し、Smalltalk・Squeak Etoys・Scratchの日本での普及に中心的な役割を果たした。

| 1987年〜 | オブジェクト指向言語Smalltalkの研究開発に従事 |

| 現在 | 青山学院大学特任教授、放送大学客員教授、津田塾大学非常勤講師 |

## Scratchとの関わり

Scratch日本語版のローカライズを担当。当初はリバースエンジニアリングにより「勝手に」日本語化を進め、その後MITメディアラボが多言語化に対応して正式な初期日本語翻訳を担当した。NHK Eテレ『Why!? プログラミング』の放送開始後、日本のScratchコミュニティのユーザー登録が7万人から急増した。

MITメディアラボのメンバーとともに、Scratchを使った**クリエイティブ・ラーニング**（創造的学習）を提唱し、日本での普及に貢献。

## 知的系譜

```
アラン・ケイ（Dynabookビジョン）
  └→ Smalltalk → Squeak → Squeak Etoys
       └→ 阿部和広（日本での普及）
            └→ Scratch日本語版
                 └→ 石原淳也（Scratch拡張エコシステム）
```

アラン・ケイのDynabookビジョン — すべての人がコンピュータを創造的なツールとして使えるようになる — を、Smalltalk → Squeak Etoys → Scratchという系譜の中で日本に根付かせた人物。石原淳也はこの系譜の延長線上で、Scratchに機械学習やハードウェア制御を接続する拡張機能群を開発している。→ [[concepts/programming-education|プログラミング教育]]

## 教育思想

### プログラミングは「表現教育」
プログラミングを産業人材育成ではなく、**子どもが自分の発想を豊かにするための表現教育**と位置づけている。

### 「教えない」教育
津田塾大学での授業では「教えないことを大切にしている」方針。学生の主体性を尊重し、相互評価やピアレビューを導入。多様な解答を許容し、試行錯誤の過程を重視。

### 創造性への危機感
「現代の子どもは何もないところから何かを作り出す力が落ちている」と指摘。小学校高学年から「他者から与えられる問題を最短で解くことに汲々として、主体性が失われていく」という問題意識。

### 構築主義（Constructionism）
シーモア・パパートの構築主義の系譜。「プログラミング的思考は料理を作るのと一緒」— 手順の最適化を試行錯誤的に求めるプロセスであり、文系理系の区別なく必要。

## 石原淳也との関わり

### 共著関係
『Raspberry Piではじめるどきどきプログラミング』（初版・増補改訂第2版）の共著者。石原淳也にとってプログラミング教育書の執筆における直接的な協働者。

### つくまなラボ
[[entities/tukumana-lab|青学つくまなラボ]]でシニア・フェロー（石原淳也はフェロー）として上位の立場で協働。

### 思想的影響
Apple本社で開催された「ｉのある教育と学習」セミナーでの「伝説のプレゼン」が、石原淳也がプログラミング教育に関心を深めるきっかけとなった（ブログ記事「[[concepts/generative-ai|生成AIによるコーディングはDynabook完成の最後のピース]]」で言及）。

### 知的系譜
阿部和広がアラン・ケイ → Smalltalk → Scratchの系譜を日本に根付かせ、石原淳也がその上にML2Scratch等の拡張エコシステムを構築するという、師弟的・協働的な関係。
