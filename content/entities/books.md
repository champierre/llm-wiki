# 著書

石原淳也の著書一覧（5冊）。

## Scratchで楽しく学ぶ アート&サイエンス

- **著者**: 石原淳也（著）、[[entities/abe-kazuhiro|阿部和広]]（監修）
- **出版社**: 日経BP
- **初版**: 2018年7月5日
- **改訂第2版**: 2021年8月26日
- **ISBN**: 978-4-82229-233-1

Scratchを使って等加速度運動、2進法、論理回路、確率、三角関数、モンテカルロ法、フラクタル図形などを学ぶプログラミング教育書。「Scratchは子ども向け」「ゲームしか作れない」「本格的なプログラミングではない」という誤解を解き、子どもから大人まで楽しめる内容。

## Raspberry Piではじめる どきどきプログラミング

- **著者**: [[entities/abe-kazuhiro|阿部和広]]・石原淳也・塩野禎隆（共著）
- **出版社**: 日経BP（「はじめるプログラミング」シリーズ）
- **初版**: 2014年

小学校中学年以上を対象に、Raspberry Piと[[entities/scratch|Scratch]]、Minecraft Piを通じて、プログラミングとものづくりの楽しさを伝える入門書。

### 増補改訂第2版

- **著者**: 阿部和広・石原淳也・塩野禎隆・星野尚（共著）
- **対応**: Raspberry Pi 3 Model B

## Raspberry PiとScratchではじめよう! 電子工作入門

- **出版社**: 日経BP
- **ISBN**: 978-4-82229-775-6

Raspberry PiとScratchを使ってLED点灯から各種センサー・外部デバイスの制御まで、電子工作の基礎を学ぶ入門書。
