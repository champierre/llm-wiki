# Facemesh2Scratch

顔メッシュ検出とScratchを接続するScratch 3拡張機能（GitHub ★41）。[[entities/ml2scratch|ML2Scratch]]から派生したプロジェクトの一つ。

## 概要

MediaPipeの顔メッシュ検出をScratch 3で利用できるようにする拡張機能。顔の468個のランドマークポイントをリアルタイムで検出し、その座標データをScratchのブロックとして利用可能にする。表情や顔の動きを使ったインタラクティブな作品を、ブロックプログラミングで作成できる。

## 技術

- **言語**: JavaScript
- **ML**: MediaPipe Face Mesh
- **プラットフォーム**: Scratch 3
